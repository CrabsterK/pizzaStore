Przykłady pokazujące używanie kolorów w formacie RGBA oraz HSLA w CSS

Kolory w formacie RGBA

Kolory w tym formacie są rozszerzeniem formatu RGB. Dodatkowo zawierają kanał alfa - przezroczystość obiektu.
Zapisywany jest jako
rgba(red, green, blue, alpha),
gdzie red, green oraz blue przyjmują wartości z przedziału 0 - 255, natomiast alpha przyjmuje wartość 0(pełna przezroczystość) - 1(całkowicie nieprzezroczysty)

Przykłady:

rgba(255, 0, 0, 0.9)

rgba(0, 255, 0, 0.5)

rgba(0, 0, 255, 0.1)

Kolory w formacie HSLA

Wartości koloru w HSLA są rozszerzeniem HLS, podobnie jak w RGBA, dodany został kanał alfa.
Wartość koloru jest definiowana w następujący sposób
hsla(hue, saturation, lightness, alpha)
gdzie

hue - barwa saturation - nasycenie lightness - jasność alpha - przezroczystość
Liczba całkowita określana w stopniach z przedziału 0 - 360 z tzw. koła barw. 0 - czerwony (Red), 120 - zielony (Green), 240 - niebieski (Blue) Wartość procentowa: 0% - szary, 100% - normalny Wartość procentowa: 0% - czarny, 50% - normalny, 100% - biały przyjmuje wartość 0(pełna przezroczystość) - 1(całkowicie nieprzezroczysty)
